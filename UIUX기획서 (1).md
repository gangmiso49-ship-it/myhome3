# 브런치 카페 'Hearth' UI/UX 기획서

## 1. 전체 디자인 콘셉트
* **Minimal & Warm (여백과 온기):** 불필요한 시각적 장식을 배제하고 넉넉한 여백(White Space)을 활용하여 시선이 콘텐츠(메뉴와 공간 사진)에 온전히 머물 수 있도록 설계합니다.
* **Magazine Editorial (매거진 스타일):** 텍스트와 이미지의 비율을 조화롭게 배치하여, 마치 감성적인 라이프스타일 매거진을 읽는 듯한 서정적인 브라우징 경험을 제공합니다.
* **Intuitive Navigation (직관적 탐색):** 사용자가 카페 정보, 메뉴, 위치 등 목적에 맞는 정보에 도달하기까지의 과정을 최소화(최대 2 Click 이내)합니다.

## 2. 메인 컬러와 보조 컬러
* **Primary Color (메인 컬러):** 
  * `Warm Ivory (#FAF8F5)`: 전체 배경색으로 사용하여 깨끗하면서도 눈이 편안한 캔버스를 제공합니다.
* **Secondary Color (보조 컬러):**
  * `Soft Beige (#E8E1D9)`: 섹션 배경, 라인, 카드형 UI의 백그라운드로 활용하여 입체감과 부드러움을 더합니다.
* **Text & Point Color (텍스트 및 포인트 컬러):**
  * `Deep Brown (#4A3B32)`: 퓨어 블랙(#000000) 대신 텍스트 컬러로 사용하여 전체적인 따뜻한 톤앤매너를 유지하면서도 가독성을 확보합니다.
  * `Brick Red (#9E503C)`: '화로(Hearth)'를 연상시키는 포인트 컬러로, CTA(Call to Action) 버튼이나 강조해야 할 타이포그래피에 제한적으로 사용합니다.

## 3. 폰트 스타일
* **영문 타이틀:** `Playfair Display` 또는 `Lora` 
  * 클래식하고 우아한 세리프(Serif) 서체를 사용하여 브랜드의 고급스러움과 아늑함을 강조합니다.
* **국문 타이틀:** `KoPub 바탕체` 또는 `나눔명조`
  * 감성적인 문구와 브랜드 스토리에 적용하여 서정적인 분위기를 연출합니다.
* **본문 (Body Text):** `Pretendard` 또는 `Noto Sans KR`
  * 가독성이 최우선인 본문과 정보성 텍스트(메뉴명, 주소, 시간 등)에는 산세리프(고딕) 서체를 사용하여 정보 전달력을 높입니다.

## 4. 메인 화면 구성
* **GNB (글로벌 네비게이션 바):** 스크롤 다운 시 배경이 반투명한 화이트로 변하는 Sticky Header 적용. 좌측 혹은 중앙에 미니멀한 로고 배치, 우측에 햄버거 메뉴(모바일) 또는 심플한 텍스트 메뉴(PC) 배치.
* **Hero Section:** 전체 화면(100vh)을 덮는 고화질 공간/브런치 연출 컷. 중앙에 타이포그래피(예: "당신의 하루를 덥혀줄 따뜻한 온기, Hearth") 배치. 부드러운 패럴랙스(Parallax) 스크롤 효과 적용.
* **Quick Overview:** 스크롤 시 브랜드 스토리 요약 -> 시그니처 메뉴 슬라이더 -> 오시는 길 미니맵이 순차적으로 페이드인(Fade-in) 되며 나타나는 구조.

## 5. 브랜드 소개 영역 구성
* **Split Layout (분할 레이아웃):** PC 기준 좌측에는 감성적인 에세이 형태의 브랜드 스토리 텍스트, 우측에는 아늑한 매장 인테리어 사진을 배치 (모바일은 상하 배치).
* **이미지 스타일링:** 사진 모서리를 약간 둥글게(border-radius: 8px~12px) 처리하여 부드러운 느낌을 부여.
* **Keyword Highlight:** '화로', '휴식', '온기' 등의 핵심 키워드에 Brick Red 컬러로 밑줄 또는 폰트 컬러 강조.

## 6. 메뉴 소개 카드 구성
* **Masonry 또는 Grid 레이아웃:** 획일적인 표 형태를 탈피하여, 이미지 크기에 리듬감을 주는 레이아웃 사용.
* **Card UI:** 
  * 썸네일 중심의 카드형 UI 채택. 
  * **Hover Effect (PC):** 카드에 마우스를 올리면 이미지가 미세하게 확대(Scale 1.05)되며 부드러운 그림자(Soft Shadow)가 생성되어 클릭을 유도.
* **Information Hierarchy:** [이미지] - [메뉴명(Bold)] - [짧은 설명(1줄)] - [가격(선택사항)] - [Tag: #Signature, #Vegan 등] 순으로 시선 이동 설계.

## 7. 친환경 가치 소개 영역 구성 (Eco-friendly Value)
* **Tone Variation:** 배경을 아주 연한 올리브 그린(#F0F2EB) 또는 질감이 느껴지는 재생지 텍스처로 변경하여 친환경 섹션임을 시각적으로 분리.
* **Iconography:** 미니멀한 라인 아이콘(Line Icon)을 활용하여 생분해 컵 사용, 유기농 식자재, 로컬 푸드 활용 등의 가치를 심플하게 표현.
* **Short Copy:** 긴 설명보다는 픽토그램과 핵심 문구 중심의 인포그래픽 형태로 구성하여 가독성 증대.

## 8. 문의하기 및 오시는 길 영역 구성
* **Contact Card:** 주소, 영업시간, 라스트 오더, 휴무일 정보, 연락처를 하나의 깔끔한 카드 형태로 그룹화하여 가독성 강화.
* **Interactive Map:** 지도 API 연동. 지도의 컬러 테마를 웹사이트의 톤(세피아/웜톤)에 맞게 커스텀 스타일링(구글맵 Snazzy Maps 활용 등)하여 디자인 통일성 유지. 마커는 브랜드 로고나 Brick 컬러 아이콘 사용.
* **Action Buttons:** '네이버 지도로 보기', '카카오맵 길찾기', '전화 걸기', '인스타그램 DM' 등 직관적인 아이콘 버튼 제공.

## 9. 모바일 반응형 구성
* **Touch-Friendly:** 모든 버튼과 링크의 터치 영역을 최소 44x44px 이상으로 확보하여 오터치 방지.
* **Stacking (수직 정렬):** PC의 좌우 다단 레이아웃을 모바일에서는 1단(1 Column) 수직 레이아웃으로 변경하여 스크롤을 통한 자연스러운 콘텐츠 소비 유도.
* **Bottom Sheet / FAB:** 중요 액션(예: 길찾기, 전화걸기)은 우측 하단에 FAB(Floating Action Button)로 고정하거나 화면 하단에 바 형태로 고정하여 언제든 접근 가능하게 설계.
* **Image Optimization:** 모바일 환경에 맞춰 로딩 속도 최적화를 위해 WebP 포맷 사용 및 반응형 이미지(srcset) 적용.

## 10. 사용자 경험을 높이기 위한 요소 (UX Enhancers)
* **Micro-interactions (마이크로 인터랙션):** 버튼 클릭 시 부드러운 색상 전환, 페이지 전환 시 부드러운 페이드인/아웃 효과 등 과하지 않은 애니메이션으로 고급스러운 감도 유지.
* **Lazy Loading:** 스크롤 위치에 따라 이미지를 로딩하여 초기 사이트 진입 속도를 단축시켜 이탈률 감소.
* **Accessibility (웹 접근성):** 이미지에 대체 텍스트(alt text) 제공, 텍스트와 배경 간의 충분한 명도 대비 확보로 모든 사용자가 편안하게 정보를 인식할 수 있도록 배려.
* **Skeleton UI:** 데이터나 이미지를 불러오는 동안 뼈대(Skeleton) 화면을 보여주어 체감 대기 시간을 줄임.
